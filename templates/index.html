Welcome to the NHS Robostangs/CSHS IDE

This is the web-based IDE (Integrated Development Environment) for programming the robot. With this system you can program the Robostang's FRC (First Robotics Competition) robot in the Python programming language. Most Python syntax and control flow still applies in this environment, however you cannot import any modules into the interpreter.

To go to the IDE, click the link in the top bar.
